## STL名称：standard template library

### STL六大部件
* 容器(Containers)
* 分配器（Alloctors）
* 算法（Algorithm）
* 迭代器（Iterators）
* 适配器（Adapters）
* 函数类（Functors）

C++标准中一共13个头文件：(分为自己常用的和不常用的,百度百科上面还有详细解释)
* 容器类
	* <vector>
	* <list>
	* <deque>
	* <stack>
	* <map>
	* <set>
	* <queue>
* 算法类
	* <alogrithm>一百多个算法函数模板所在文件
	* <numeric>少量进行序列运算的函数模板
	* <functional> 模板类函数所在
* 迭代器
	* <iterator>
* 工具支持（为其他模板）
	* <utility>
* 内存支持（为其他模板）
	* <memory.h>
